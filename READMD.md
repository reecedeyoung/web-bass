# Web Bass
this  project is a simple web app that provides a synthesized bass guitar emulation for users to play. It contains an animated svg bass and can be controlled via midi. 

## Content

### Infra 
this dir contains all of the IAC for this project. Project is hosted on AWS and uses terraform IAC

### Web-Base
this dir contains the frontend code for the web app. The front end is a vite react spa using typescript and component ui.
